To get up and running:

1. Make sure you have the latest Java installed: http://docs.oracle.com/javase/8/docs/technotes/guides/install/install_overview.html#A1096936
2. Download Elasticsearch (windows version): https://www.elastic.co/downloads/elasticsearch
3. Run batch file on step 2. This MUST be running.
4. (Optional, highly recommended) Install Kibana (windows version): https://www.elastic.co/downloads/kibana
5. Skip step 2. Run batch file on step 3. This MUST be running.

Elasticsearch is now installed.
To verify: http://localhost:9200

To run Kibana: http://localhost:5601/app/kibana#/dev_tools
Note, Kibana is a test environment where you can run all of your commands against Elasticsearch.

To install initial CRMLS data. Build and run this application.
